Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: commit all files in the working branch.
  • git commit -m "text-message-about-this-commit": commit all the staged changes to the remote repository.
  • The commit message should be short descriptive messages of the changes implemented. See Git Commit Guidelines.
